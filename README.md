# Rose Fashion

Rose Fashion is a fully integrated e-commerce platform for selling clothes online, built using Laravel. It features an instant shopping cart, a secure payment system, and a comprehensive admin dashboard for managing products and orders.

---

## Tech Stack

* PHP
* Laravel
* Blade Templates
* HTML5
* CSS3
* JavaScript

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/laravel-ecommerce-store.git
```

Navigate to the project folder:

```bash
cd laravel-ecommerce-store
```

Install dependencies:

```bash
composer install
```

Copy environment file:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Run the development server:

```bash
php artisan serve
```
